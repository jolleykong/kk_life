> # The complete list of new features in MySQL 8.0
>
> [January 28, 2019](https://mysqlserverteam.com/the-complete-list-of-new-features-in-mysql-8-0/)[MySQL](https://mysqlserverteam.com/category/mysql/)[Geir Hoydalsvik](https://mysqlserverteam.com/author/geir/)
>
> There are over 300 new features in MySQL 8.0. The MySQL Manual is very good, but verbose. This is a list of new features in short bullet form. We have tried very hard to make sure each feature is only mentioned once. Note the [similar list for MySQL 5.7](http://www.thecompletelistoffeatures.com/).
>
> Please download MySQL 8.0 from [dev.mysql.com](https://dev.mysql.com/downloads/) or from the MySQL [Yum](https://dev.mysql.com/downloads/repo/yum/), [APT](https://dev.mysql.com/downloads/repo/apt/), or [SUSE](https://dev.mysql.com/downloads/repo/suse/) repositories.
