- 基础命令

  - select：选择数据库

    ```
    select 1
    ```

    

  - set：创建键

    ```
    set <key> <value>
    ```

  - del：删除键

    不支持通配符

    ```
    del <key>[<key>]
    ```

  - Keys：获得符合规则的key名列表

    会遍历redis中所有键

    ```
    keys <expr>
    ```

  - exists：判断一个键是否存在

    ```
    exists <key>
    ```

  - type：查看键的数据类型

    ```
    type <key>
    ```
